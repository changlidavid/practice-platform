# Reorder Digits Descending

Write a function `reorder(number)` that rearranges the digits of `number` from largest to smallest.

The sign of the result should match the sign of the input.

## Function Signature

`reorder(number)`

## Notes

- `number` is an integer.
- For positive inputs, return a positive integer.
- For negative inputs, return a negative integer whose digits are also ordered from largest to smallest.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
