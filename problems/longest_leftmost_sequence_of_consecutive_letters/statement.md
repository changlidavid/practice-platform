# Longest Leftmost Sequence of Consecutive Letters

Write a function `longest_leftmost_sequence_of_consecutive_letters(word)` that returns the longest substring of `word` whose letters are consecutive in the alphabet.

If there is more than one longest substring, return the leftmost one.

## Function Signature

`longest_leftmost_sequence_of_consecutive_letters(word)`

## Notes

- `word` contains only lowercase letters.
- A consecutive sequence increases by 1 in letter order, such as `abc` or `rstuv`.
- If no two adjacent letters are consecutive, return the leftmost single letter.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
