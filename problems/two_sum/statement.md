# Two Sum

Given a list of integers `nums` and an integer `target`, return the indices of the two numbers that add up to `target`.

Assume exactly one valid answer exists, and you may not use the same element twice.

Return the indices in any order.

## Function Signature

`two_sum(nums, target)`

## Notes

- `nums` contains integers.
- Return a list of two indices.
- Hidden tests include additional edge cases.
