# Filtered Sequence

Write a function `filtered_sequence(L, n)` that keeps from `L` only those elements that belong to a contiguous run of length at least `n`.

All elements in such a run have the same value.

## Function Signature

`filtered_sequence(L, n)`

## Notes

- `L` is a list of integers.
- Runs are contiguous. Equal values separated by other values belong to different runs.
- Return the filtered list in the same order as in `L`.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
