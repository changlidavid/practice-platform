# Collapse Consecutive Digits

Write a function `collapse(number)` that collapses each maximal run of identical consecutive digits in `number` into a single digit.

The sign of the result should match the sign of the input.

## Function Signature

`collapse(number)`

## Notes

- `number` is an integer.
- Consecutive repeated digits are reduced to one copy.
- If the input is negative, the returned value should also be negative unless the result is `0`.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
