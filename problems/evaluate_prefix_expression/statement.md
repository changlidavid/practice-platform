# Evaluate Prefix Expression

Write a function `evaluate_prefix_expression(expression)` that evaluates a syntactically correct prefix expression.

For this problem:

- operands are integers
- operators are `+`, `-`, `*`, and `/`
- tokens are separated by spaces

## Function Signature

`evaluate_prefix_expression(expression)`

## Notes

- Assume the input expression is syntactically correct.
- Assume division is never applied with a zero denominator.
- Return the numeric result of the expression.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
