# Is Heterosquare

Write a function `is_heterosquare(square)` that returns `True` if `square` is a heterosquare and `False` otherwise.

A heterosquare of order `n` is a square arrangement of the integers `1` to `n**2` such that all row sums, column sums, and diagonal sums are different.

## Function Signature

`is_heterosquare(square)`

## Notes

- `square` is represented as a list of lists of integers.
- A valid heterosquare must use each integer from `1` to `n**2` exactly once.
- Return a Boolean value.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
