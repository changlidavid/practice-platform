# Remove Consecutive Duplicates

Write a function `remove_consecutive_duplicates(word)` that returns a new string formed by collapsing each run of consecutive identical characters into a single character.

## Function Signature

`remove_consecutive_duplicates(word)`

## Notes

- `word` is a string.
- Return the result as a string.
- The result should keep the original character order.
- Hidden tests reuse additional cases from the original doctest problem.
