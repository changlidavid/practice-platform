# Good Subsequences

Given a word `word`, return the list of all good subsequences of `word`, without duplicates, in lexicographic order.

A good subsequence is a string that:

- is obtained by deleting zero or more characters from `word`
- contains no repeated letters

## Function Signature

`good_subsequences(word)`

## Notes

- Return a list of strings.
- Include the empty string `''`.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
