# Average of Digits Common To

Write a function `average_of_digits_common_to(*numbers)` that returns the average of the digits that appear in every provided non-negative integer.

Each common digit is counted at most once, even if it appears multiple times within a number.

## Function Signature

`average_of_digits_common_to(*numbers)`

## Notes

- If no numbers are provided, return `-1`.
- If the numbers have no digit in common, return `-1`.
- Otherwise return the arithmetic mean of the common digits.
- `public_examples.json` is display-only.
- `hidden_tests.json` is the full official evaluation set.
